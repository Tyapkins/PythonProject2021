Ежедневник
===========
Первая из функций данного приложения -- подобие ежедневника, т.е. сборник напоминаний. Пользователь может вводить напоминания, выделяя особо важные. Данная информация может сохраняться в файл, который затем можно считывать и, соответственно, сохранять введённые напоминания даже при отключении приложения. Пользователю выдаётся информация о всех введённых им напоминаниях, о ближайшем "деле". Неактуальные напоминания (т. е. те, дата которых прошла) предлагаются к удалению: впрочем, пользователь может их оставить, дабы поменять затем дату.
